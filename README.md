# Intent-to-Auditable-Trust-Object (IATO-V2)

A production-oriented FastAPI service for **banknote authenticity prediction** using a trained scikit-learn classifier.

This repository provides a lightweight inference API, health endpoint, and OpenAPI documentation suitable for controlled deployment and integration into security-conscious environments.


## 1) Solution Overview

### Core capabilities
- Serve a trained model for real-time classification of banknote records.
- Expose predictable REST endpoints for health checks and prediction.
- Provide OpenAPI/Swagger docs for interface governance and integration.

### API surface
- `GET /` – service metadata/welcome response.
- `GET /health` – liveness/health endpoint.
- `POST /predict` – model inference endpoint using 4 numeric features:
  - `variance`
  - `skewness`
  - `curtosis`
  - `entropy`

Implementation details and endpoint behavior are in `Fast_Api/README.md` and code under `Fast_Api/`.

## 2) Tooling, Runtime, and Reference Specs

### Primary tooling map
- **Framework**: FastAPI (`fastapi`) for typed HTTP APIs.
- **ASGI server**: Uvicorn (`uvicorn`) for local and container runtime serving.
- **ML runtime**: scikit-learn (`sklearn`) model artifact loaded for inference.
- **Serialization / model artifact**: `pickle` (`classifier.pkl`) for model persistence.
- **Validation**: Pydantic models for request schema integrity.

### Interface and protocol references
- **API contract**: OpenAPI 3.x (auto-generated by FastAPI).
- **Transport**: HTTP/1.1 JSON request/response patterns.
- **Operational endpoint references**:
  - Swagger UI: `/docs`
  - ReDoc: `/redoc`

> Note: This project is a reference implementation for inference and integration. Production hardening (e.g., authN/authZ, secrets management, rate limiting, WAF, tamper-proof audit pipelines) should be added in the deployment environment.

## 4) Quick Start 

### Prerequisites
- Python 3.9+ recommended.
- `pip` available.

### Install and run
```bash
cd Fast_Api
pip install -r requirements.txt
uvicorn Fast_Api.app:app --host 0.0.0.0 --port 8000 --reload
```

### Verify service
```bash
curl http://127.0.0.1:8000/health
```

Expected: JSON health status response.

### Run a sample prediction
```bash
curl -X POST 'http://127.0.0.1:8000/predict' \
  -H 'Content-Type: application/json' \
  -d '{
    "variance": 3.6216,
    "skewness": 8.6661,
    "curtosis": -2.8073,
    "entropy": -0.44699
  }'
```

### API docs
- Swagger UI: `http://127.0.0.1:8000/docs`
- ReDoc: `http://127.0.0.1:8000/redoc`
